#  Python Quiz App

A console-based quiz application built using Python and JSON.  
It displays multiple choice questions, checks answers and calculates final score.

###  Features
- 10 multiple-choice questions
- Score calculation (10 points per correct answer)
- Shows correct answer if the user is wrong
- JSON file for easy question editing
